# Administración de Sistemas

Esta sección recopila documentación técnica relacionada con la **administración de sistemas**, redes y servicios.  

El enfoque es **práctico, aplicado y seguro**, orientado a mantener la estabilidad, disponibilidad y eficiencia de infraestructuras IT.

---

## Objetivos de la sección

- Administrar sistemas y servicios de manera eficiente y segura
- Resolver incidencias y problemas reales en producción
- Mejorar la estabilidad, disponibilidad y resiliencia de sistemas
- Documentar procedimientos técnicos y buenas prácticas operativas
- Servir como referencia para profesionales de TI y administradores de sistemas

Se prioriza la **claridad y aplicabilidad práctica** por encima de configuraciones de laboratorio.

---

## Áreas cubiertas

### Bases de Datos
- Diseño conceptual y lógico.
- Administración avanzada.
- Optimización del rendimiento de las operaciones
- Seguridad y alta disponibilidad de los datos

### Sistemas Operativos
- Implantación y despliegue de sistemas operativos  
- Administración de usuarios, permisos, procesos y servicios  
- Configuración y mantenimiento de entornos Linux y Unix-like
- Windows Server y Active Directory

### Redes y Servicios
- Planificación, Configuración y administración de redes  
- Servicios de red como DHCP, DNS y routing  
- Monitoreo, troubleshooting y optimización de conectividad  

### Seguridad y Alta disponibilidad
- Mantenimiento de la disponibilidad y resiliencia de sistemas  
- Monitorización de servicios y control de accesos  
- Buenas prácticas de seguridad operativa  

### Hardware
- Configuración, diagnóstico y mantenimiento de equipos físicos  
- Monitorización de recursos y resolución de incidencias de hardware  

### Aplicaciones web
- Lenguaje de marcas
- Implantación de aplicaciones web (Wordpress)

### Programación
- Introducción a Python
- Programación orientada a objetos
- Bash scripting

---

## Enfoque profesional

El contenido de esta sección está diseñado para:

- Entornos de producción y operación real
- Resolución de problemas y optimización de sistemas
- Aplicación de buenas prácticas de administración y mantenimiento
- Preparar al administrador para gestionar infraestructuras completas con seguridad y eficiencia

---

## Relación con la ciberseguridad

La administración de sistemas siempre se aborda considerando:

- Principios de mínimo privilegio y control de accesos  
- Actualizaciones y mantenimiento continuos  
- Reducción de superficie de ataque y riesgos operativos
